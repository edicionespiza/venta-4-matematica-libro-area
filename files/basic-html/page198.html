Suceso
                Es un hecho o acontecimiento que puede ocurrir o no.

              Ejemplos:

                Que le salga una pelota amarilla a Sergio.              Que el domingo llueva en Lima.
                Lanzar un dado y que salga en el número 4.              Lanzar una moneda y que salga sello.

              Espacio muestral (    )

               Es el conjunto de todos los datos posibles de un experimento aleatorio.
              Ejemplos:

                El espacio muestral de la máquina de pelotas:              = {roja, amarilla, verde, azul}

                El espacio muestral al lanzar una moneda:             = {cara, sello}


                                                          Aprende
                La probabilidad de un suceso es mayor o igual que cero, pero menor o igual que
                1, es decir: 0 ≤ P(A) ≤ 1
                Un suceso imposible nunca ocurre, es decir:  P(A) = 0
                Un suceso seguro siempre ocurre, es decir:          P(A) = 1
